# Your Caricature

AI 기반 캐리커처 생성 서비스의 UI 데모 프로젝트입니다.

## 주요 기능 (UI 데모)
- 메인 화면: 사진 업로드 및 스타일 선택
- 결과 화면: 변환된 캐리커처 확인, 저장 및 공유

## 시작하기
`index.html` 파일을 브라우저에서 열면 바로 확인할 수 있습니다.

## 배포 방법 (GitHub Pages)
1. 이 저장소를 GitHub에 업로드합니다.
2. Settings > Pages 메뉴로 이동합니다.
3. Branch를 `main`으로 설정하고 `Save`를 누르면 잠시 후 웹사이트 주소가 생성됩니다.
